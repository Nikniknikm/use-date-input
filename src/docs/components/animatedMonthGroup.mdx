# AnimatedMonthGroup

The `AnimatedMonthGroup` component animates it's children.

## Props
## AnimatedMonthGroup Props Table
| Prop Name | Type | Is Required | Default Value | Description |
|-|-|-|-|-|
| children| `element`| optional| | Children|
| firstColumnRef| `object`| optional| | Ref to the first column|
| groupRef| `object`| optional| | Ref to the MonthGroup|
| visibleFromDate| `object`| optional| | Visible from date, used to control animation between columns and MonthGroup|
