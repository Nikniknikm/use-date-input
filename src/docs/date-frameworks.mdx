# Date Frameworks

`use-date-input` provides an adapter API that enables it to work with any date framework.

The adapter interface is provided to `use-date-input` components through the `adapter` prop.

Adapters have been created for all the popular Date frameworks. To keep the bundle size low and to let the user manage versioning, the provided adapters 
require peer dependencies to specify the date API.

- <a href="https://date-fns.org/" target="__blank__">date-fns</a>
- <a href="https://day.js.org/" target="__blank__">dayjs</a>
- <a href="https://moment.github.io/luxon/" target="__blank__">luxon</a>
- <a href="https://momentjs.com/" target="__blank__">moment</a>
- ... or write your own

##  With date-fns

Install the dependencies with

```shell script
yarn add @date-fns
yarn add @use-date-input/date-fns-adapter
```
OR

```shell script
npm install -s @date-fns
npm install -s @use-date-input/date-fns-adapter
```

Import the adapter into your component

```javascript
import { adapter as dateFnsAdapter } from "@use-date-input/date-fns-adapter";
```

Use the `adapter` prop with the `use-date-input` components.

<Playground>
{ () => {
    // import { Calendar } from "@use-date-input/core";
    // import { adapter as dateFnsAdapter } from "@use-date-input/date-fns-adapter";
    return (
        <Calendar
            adapter={dateFnsAdapter}
            numOfColumns={2}
            numOfVisibleMonths={2}
        />
    );
}}
</Playground>

##  Week Offset

Date frameworks use different week offsets, so you can customize on `use-date-input` components through the use of the `weekOffset` prop

<Playground>
{ () => {
    // import { Calendar } from "@use-date-input/core";
    // import { adapter as luxonAdapter } from "@use-date-input/luxon-adapter";
    return (
        <Calendar
            adapter={luxonAdapter}
            numOfColumns={2}
            numOfVisibleMonths={2}
            weekOffset={6}
        />
    );
}}
</Playground>

##  Write Your Own Adapter

To write your own adapter or customize the existing adapters, you need to build this interface and provide it
through the `adapter` prop.

```javascript
const adapter = ({ weekOffset = 0 } = {}) => ({
  addDays,
  addMonths,
  addWeeks,
  addYears,
  createDate,
  daysInMonth,
  dayOfWeek,
  diffInDays,
  diffInMonths,
  endOfMonth,
  endOfWeek,
  endOfYear,
  format,
  getDateFormat,
  isAfter,
  isBefore,
  isSameDay,
  isSameMonth,
  isValid:
  set,
  startOfMonth,
  startOfWeek,
  startOfYear,
  subtractDays,
  subtractMonths,
  subtractWeeks,
  subtractYears
});
```

For the most cases, this just requires mapping these callbacks to the equivalent date framework call.  
  
For further examples, refer to the provided adapters code, such as 
[@use-date-input/dayjs-adapter](https://github.com/mark-tate/use-date-input/blob/master/packages/date-fns-adapter/src/index.js).
